# 任务二：剪映AI音色克隆与配音产出说明

## 一、克隆步骤概要
1.  **准备素材**：打开剪映PC版，新建项目，导入一段参考视频作为底素材。
2.  **首次朗读与音色录制**：
    - 点击「文本」→「新建文本」，输入任意测试文本；
    - 选中文本，点击「文本朗读」，在音色列表中选择「AI克隆」；
    - 按照引导录制一段1分钟左右的本人声音（清晰、无背景噪声），完成音色克隆；
    - 克隆成功后，将该音色保存为自定义音色（命名为「我的音色」）。
3.  **文案导入与配音**：
    - 将DeepSeek大模型生成的文案导入剪映，拆分为多段文本添加到视频时间轴；
    - 选中所有文本，点击「文本朗读」，选择刚刚克隆的自定义音色；
    - 系统自动匹配语速、语调，生成与文本对应的配音。
4.  **导出与存储**：
    - 完成配音后，预览视频效果，确认无误后导出；
    - 导出文件存储至夸克网盘，提供可访问链接。

## 二、导出文件格式
- **文件格式**：MP4
- **编码标准**：H.264视频编码 + AAC音频编码
- **分辨率**：1080p（1920×1080）
- **帧率**：30fps
- **音频参数**：48kHz采样率，立体声
